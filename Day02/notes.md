Learnt how to view, search and troubleshoot files using Linux commands.

# Commands Learned

| Command | Purpose | Example |
|----------|---------|---------|
| cat | Display entire file | cat app.log |
| less | Read large files page by page | less app.log |
| head | Show first few lines | head -5 app.log |
| tail | Show last few lines | tail -10 app.log |
| tail -f | Monitor log file in real time | tail -f app.log |
| grep | Search text inside files | grep ERROR app.log |
| find | Find files/directories | find . -name "*.log" |


# Key Learnings

- `cat` is useful for small files.
- `less` is preferred for large files.
- `head` shows the beginning of a file.
- `tail` shows the end of a file.
- `tail -f` is used to monitor logs in real time.
- `grep` searches text inside files.
- `find` searches recursively for files and directories.
